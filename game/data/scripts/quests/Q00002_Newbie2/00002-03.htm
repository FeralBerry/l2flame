Бакалейщик Папума:

В магазинах бакалеи вы можете найти различные расходники, которые нужны для охоты на монстров.
Например для улучшения и ускорении охоты нужны заряды Души и Духа, различные снадобья для восстановления здоровья и маны, для начальных уровней вы можете их покупать в любом магазине бакалеи. И так же для лучников магазин бакалеи долгое время является поставщиком стрел для лука и арбалета.
Для гномов магазин бакалеи нужен почти всегда, потому что в нем можно приобрести расходники для создания различных вещей.
